# Game Title
Beneath the Strings

## Narative
A lost puppet awakens in a cavern of shattered gears, with only a narrow tunnel leading deeper into the unknown.
As a crystal chandelier crashes, brass symphonies fill the air. From the darkness, a colossal machine emergesâ€”a haunting reflection of the puppet itself.

"Is this me?"

In a monochrome, hand-drawn world, face a gripping 2D boss fight where fate and identity intertwine. Presented by The Puppeteers.

## Controls
A : Move Left

D : Move Right

Space key : Jump

Left shift : Dash

Left mouse click : Attack
